# DataLens Pro – Your Open‑Source Tableau Replacement 🚀

## Overview
DataLens Pro is a next‑generation, **premium‑grade** data‑visualisation web app built with **Streamlit** and **Plotly**.  It empowers analysts, data scientists, and business users to upload Excel or CSV files and instantly generate **50+ interactive dashboards** and **50+ stunning visualisations** – all without any licensing fees.

## Why DataLens Pro?
- **Zero cost, 100 % open source** – No hidden fees or row limits.
- **Lightning‑fast insights** – Upload a file and see a full suite of KPI cards, statistical summaries, time‑series, clustering, and more within seconds.
- **Rich, colourful UI** – Glass‑morphism inspired styling, vibrant gradients, and smooth micro‑animations make every interaction feel premium.
- **One‑click export** – Download a beautiful, self‑contained HTML report that mirrors the full dashboard experience.
- **Extensible** – Built on Python, you can drop‑in any additional library or custom visual for deeper analytics.

## Core Features
- **Ultra‑fast data loading** with caching for CSV and Excel.
- **Automatic type detection** – dimensions, measures, and dates are identified for you.
- **Global sidebar filters** – slice data across **all** dashboards with multi‑select, range sliders, and date pickers.
- **KPI Overview** – instant cards showing totals, averages and deltas for top measures.
- **Statistical Summary** – descriptive statistics table with gradient highlighting.
- **Missing‑data analysis** – heatmap of null percentages.
- **Correlation heatmaps** – interactive matrix and absolute correlation views.
- **Distribution suite** – histograms, box‑plots and violin plots for every measure.
- **Categorical exploration** – bar & pie combos for low‑cardinality dimensions.
- **Time‑Series explorer** – period aggregation with dual‑axis sum/mean charts.
- **Advanced analytics** – PCA visualisation and K‑Means clustering with 3‑D scatter.
- **Custom Builder** – pick any X/Y axis, choose chart type (Bar, Line, Scatter, Box, Violin, Area) and generate on the fly.
- **Auto‑Graph Gallery** – a curated collection of 50+ pre‑built charts covering every analytical angle.

## Seamless Workflow
1. **Upload** – Drag‑and‑drop one or multiple CSV/Excel files.
2. **Auto‑analyse** – The app instantly recognises dimensions, measures, and dates.
3. **Filter** – Use the sidebar to narrow down data; filters apply globally.
4. **Explore** – Switch between tabs for KPI, distributions, correlations, time‑series, advanced analytics, and the auto‑graph gallery.
5. **Export** – Click **Export Dashboard (HTML)** to download a colourful, interactive HTML report that can be shared offline or embedded anywhere.

## Getting Started (No Code Needed)
- Clone the repository.
- Install the required Python packages (`streamlit`, `pandas`, `plotly`, `scikit‑learn`, etc.).
- Run the app with `streamlit run ap.py`.
- Open the provided URL in your browser and start visualising.
